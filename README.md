# aidware.dev

Website for **aidware** — a boutique software and AI studio founded by Aida Ghergus, building production-ready AI applications for legal and real estate clients.

Built with [Astro](https://astro.build), Tailwind CSS and React islands. Deployed on Vercel.

## Commands

```sh
npm install       # install dependencies
npm run dev       # start dev server at localhost:4321
npm run build     # type-check and build to ./dist
npm run preview   # preview the production build
```

## Structure

- `/` — client-facing landing page (hero, services, selected work, studio, contact CTA)
- `/projects` — full project portfolio
- `/about` — founder background, experience and skills
- `/contact` — contact form and links
- `/cv` — recruiter-focused online CV with print/PDF support
